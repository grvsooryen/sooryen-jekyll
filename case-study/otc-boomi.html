partnered with

Oriental Trading Company is the nation’s largest direct merchant of value-priced party supplies, arts and crafts, toys and novelties, and a leading provider of school supplies and affordable home décor and giftware. Acquired by Berkshire Hathaway in 2012, OTC has been recognized as one of the Top 50 Internet Retailers and one of the Top 50 Catalog Companies.

Challenge

Sooryen was approached by Oriental Trading Company (OTC) to integrate their Order Management System (OMS) with Target.com, one of their biggest customers, for a seamless Order Management experience across the two companies.

However, OTC’s OMS documents were output in JSON (JavaScript Object Notation) data interchange format (Json 846, Json 855, Json 856, Json 865) and needed to be accessed by a Web Service call, while Target.com required documents (EDI 846, EDI 855, EDI 856, EDI 865) in the ASC (Accredited Standards Committee) X12 format through an AS2 server. Similarly, EDI data coming from Target.com (EDI 850, EDI 860) needed to be translated for OTC (Json 850, Json 860).

Implementation

Since the EDI formats needed to be transformed at runtime between the systems at Oriental Trading Company and Target, the Sooryen Technologies team implemented the format transformation using a process deployed on Dell Boomi Atom Cloud. Dell Boomi provides a Graphical User Interface (GUI) to develop Atoms, which can then be deployed directly to their cloud. Also Dell Boomi offers a REST API to support Web Service calls with OTC and an AS2 server to support the interface with Target.com. The Atoms developed by Sooryen transform formats from JSON to ASC X12 and also from ASC X12 to JSON enabling Electronic Data Interchange in both directions. This project was rapidly completed in about 2 months.
